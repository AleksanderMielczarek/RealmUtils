# Realm Utils

Utilities for [Realm](https://realm.io/docs/java/latest/).

## Usage

Add it in your root build.gradle at the end of repositories:

```groovy
allprojects {
	repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```
